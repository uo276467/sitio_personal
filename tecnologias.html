Tecnologías que me interesan

Aquí dejo algunas tecnologías interesantes con las que he trabajado o me gustaría trabajar.

GraphQL

GraphQL es un lenguaje de consulta y un runtime desarrollado por Facebook que permite a los clientes pedir exactamente los datos que necesitan de una API, ni más ni menos. A diferencia de REST, reduce la sobrecarga de múltiples peticiones y hace más eficiente la comunicación entre cliente y servidor, ofreciendo flexibilidad y mejor rendimiento en el consumo de datos.

React

React es una librería de JavaScript desarrollada por Meta para construir interfaces de usuario de forma declarativa y eficiente. Se basa en componentes reutilizables y en un sistema de actualización reactivo que facilita la creación de aplicaciones web dinámicas y escalables.

Tailwind CSS

Tailwind CSS es un framework de utilidades para CSS que permite diseñar interfaces rápidamente usando clases predefinidas directamente en el HTML. Ofrece gran flexibilidad y evita escribir hojas de estilo personalizadas extensas, acelerando el desarrollo y manteniendo el código más limpio.

Docker

Docker es una plataforma que permite empaquetar aplicaciones y sus dependencias en contenedores ligeros y portables. Estos contenedores garantizan que el software se ejecute de manera consistente en cualquier entorno, facilitando el despliegue, la escalabilidad y la integración en flujos de desarrollo modernos.
